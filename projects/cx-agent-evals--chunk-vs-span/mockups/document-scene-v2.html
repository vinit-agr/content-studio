What changed vs v1

  • Normalize output is now four .md files, one per input — preserves the many-to-many mapping.
  • Each file card shows annual-report.md, case-study.md, blog-post.md, meeting-notes.md with a small folded-corner .md icon and a "from PDF / from DOCX / …" source hint.
  • A cursor enters from the bottom-right, moves to the first file, clicks — ring pulse confirms interaction.
  • The clicked file scales up from its own position into the full structured-markdown card (origin top-right, so it grows out of where the cursor clicked).
  • Other three files dim and fade when the click lands, so focus stays on the selected document.

Beat timing (13s loop, 10.5s scene)

  • 0.0–3.2s — 4 format icons cascade in
  • 3.2–5.0s — flow right into normalize box
  • 5.0–6.0s — 4 .md files emerge one-by-one on the right
  • 6.8–8.0s — cursor enters, moves to annual-report.md
  • 8.0s — click (ring pulse)
  • 8.2–10.2s — clicked file expands to content; heading, paragraph, highlight, bullets, LLM-native badge cascade in
  • 10.2–10.5s — hold · then loop reset
